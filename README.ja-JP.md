# Facet - Minecraft ブロック数カウントとアウトライン補助 Mod

[English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja-JP.md)

公式プロジェクトページ：[Modrinth](https://modrinth.com/project/lW1ulmxe) | [CurseForge](https://legacy.curseforge.com/minecraft/mc-mods/facet)

Facet は、Minecraft のブロック数カウント、ブロックアウトライン、距離確認、マンハッタン経路の測定、グラフィティによる目印を補助する軽量な Fabric クライアントサイド Mod です。建築、床張り、区画整理、ルート設計、大規模プロジェクトでブロックを数えたり目印を付けたりしやすくしながら、お気に入りのリソースパックのテクスチャはそのまま使えます。

## 理念と開発意図

Facet の目的はシンプルです。ブロックを数えやすくしつつ、プレイヤーが好きなテクスチャを犠牲にしないことです。多くの Minecraft リソースパックは、ブロックのテクスチャ自体に恒久的な境界線を追加します。これはカウントには便利ですが、すべてのブロックの見た目を常に変えてしまいます。Facet は境界線をクライアント側の描画オーバーレイとして重ねるため、バニラテクスチャ、カスタムリソースパック、シェーダー向けのテクスチャを変更したり上書きしたりしません。

ブロックを数えたいときは、キー割り当てで全ブロックのアウトライン、対象ブロックのアウトライン、距離 HUD を切り替えられます。必要がなくなったらオフにして、普段の見た目のままプレイを続けられます。長い距離を測る場面では、マンハッタン経路インジケーターがプレイヤー足元のブロックから対象ブロックまでの X、Z、Y 方向の経路を示し、HUD には X、Z、Y と合計マンハッタン距離が表示されるため、経路上のブロック数をひと目で確認できます。

グラフィティ機能は、建築中にブロックへ目印を付けるための機能で、探索時の道しるべとしても使えます。対応するブロック面に照準を合わせてグラフィティホイールを開くと、マーカーを選択できます。グラフィティの保存と描画はクライアント側だけで行われるため、ほかのプレイヤーには見えません。印を付けたブロックが移動または破壊されると、そのグラフィティも削除されます。

## 機能概要

- 自分の Minecraft リソースパックのテクスチャを維持したまま使える、切り替え可能なブロックアウトライン
- 通常のインタラクション範囲外の対象も示せる、対象ブロック用のネオンアウトライン
- X、Z、Y と合計マンハッタン距離を表示する距離カウント HUD
- 経路区間を色分けして見やすく示すマンハッタン経路インジケーター
- 建築や探索に使えるクライアント側のグラフィティマーカー（ほかのプレイヤーには表示されません）
- アウトラインの不透明度、幅、対象ブロックアウトラインのスタイルなどのクライアント側設定

## キー割り当て

- 全ブロックアウトラインの切り替え
- 対象ブロックアウトラインの切り替え
- ブロック距離カウント HUD とマンハッタン経路表示の切り替え
- グラフィティホイールを開く（デフォルトは `G`、`1`～`4` またはマウスでマーカーを選択）
