# Facet v1.0.3

## Compatibility

- Minecraft 26.1: `Facet-Fabric-1.0.3-26.1.jar`
- Minecraft 26.2: `Facet-Fabric-1.0.3-26.2.jar`
- Loader: Fabric Loader 0.19.3 or newer
- Runtime: Java 25 or newer

## Highlights

- Adds material-colored block outlines for easier block counting.
- Adds a neon outline for the targeted block.
- Keeps target feedback visible outside the normal interaction range.
- Adds a three-axis distance HUD and Manhattan-distance path indicator.
- Standardizes release artifact names as `Facet-Fabric-<project_version>-<minecraft_version>.jar`.

## Installation

1. Download the jar that matches your Minecraft version.
2. Put the jar in the Minecraft `mods` folder.
3. Start Minecraft with a Fabric profile.

## Notes

- This is a client-only mod.
- Mod Menu is optional but recommended for configuration.
