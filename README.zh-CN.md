# Facet - Minecraft 方块计数与描边辅助 Mod

[English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja-JP.md)

Facet 是一个轻量级 Minecraft Fabric 客户端 Mod，专为建筑、铺地、圈地、路线规划和大型工程中的方块计数而设计。它通过可随时切换的方块描边、目标方块高亮、距离 HUD 和曼哈顿路径指示器，让你在不替换材质包的前提下快速看清每个方块的位置和数量。

## 理念与初衷

Facet 的开发初衷很简单：让数方块更清楚，同时不牺牲玩家自己喜欢的材质。很多资源包会直接在材质贴图上加入边框，虽然能辅助计数，但也会永久改变所有方块的视觉风格。Facet 选择把边框作为客户端渲染效果叠加到方块上，因此不会修改或覆盖原版材质、光影适配材质，也不会影响你正在使用的资源包贴图。

当你需要计数时，可以用快捷键打开全体方块描边、目标方块描边或距离 HUD；不需要时随时关闭，世界会回到你熟悉的材质表现。对于需要按轴统计距离的场景，曼哈顿路径指示器会标记从玩家脚下到目标方块的 X、Z、Y 路径，并在 HUD 中统计 X、Z、Y 和总曼哈顿距离，方便你快速确认路径上的方块数量。

## 功能概览

- 可切换的方块材质描边，保留你自己的 Minecraft 资源包材质不变
- 目标方块霓虹描边，交互范围外的目标方块也能继续标识
- 距离计数 HUD，显示 X、Z、Y 三个方向和总曼哈顿距离
- 曼哈顿路径指示器，用不同颜色清晰标记路径段
- 描边透明度、描边宽度、目标描边样式等客户端配置
