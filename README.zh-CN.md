# Facet - Minecraft 方块计数与描边辅助 Mod

[English](./README.md) | [简体中文](./README.zh-CN.md) | [日本語](./README.ja-JP.md)

官方项目页面：[Modrinth](https://modrinth.com/project/lW1ulmxe) | [CurseForge](https://legacy.curseforge.com/minecraft/mc-mods/facet)

Facet 是一个轻量级 Minecraft Fabric 客户端 Mod，专为建筑、铺地、圈地、路线规划和大型工程中的方块计数与标记而设计。它通过可随时切换的方块描边、目标方块高亮、距离 HUD、曼哈顿路径指示器和涂鸦标记，让你在不替换材质包的前提下快速看清每个方块的位置和数量。

## 理念与初衷

Facet 的开发初衷很简单：让数方块更清楚，同时不牺牲玩家自己喜欢的材质。很多资源包会直接在材质贴图上加入边框，虽然能辅助计数，但也会永久改变所有方块的视觉风格。Facet 选择把边框作为客户端渲染效果叠加到方块上，因此不会修改或覆盖原版材质、光影适配材质，也不会影响你正在使用的资源包贴图。

当你需要计数时，可以用快捷键打开全体方块描边、目标方块描边或距离 HUD；不需要时随时关闭，世界会回到你熟悉的材质表现。对于需要按轴统计距离的场景，曼哈顿路径指示器会标记从玩家脚下到目标方块的 X、Z、Y 路径，并在 HUD 中统计 X、Z、Y 和总曼哈顿距离，方便你快速确认路径上的方块数量。

涂鸦功能可在施工时快速标记方块，也可以在探险途中用作路标。将准星对准支持的方块表面并打开涂鸦轮盘，即可选择标记。涂鸦完全由客户端本地保存和渲染，其他玩家看不到；如果被标记的方块被移动或摧毁，对应涂鸦也会随之清除。

## 功能更新

### Unreleased

- 方块描边现在会根据 MapColor 自适应调整色相、饱和度和亮度，让暖色、冷色、白色、深色和中性色边界更清晰
- 作用范围内的目标方块使用荧光绿色完整轮廓，命中面以全局同步颜色旋转色相；范围外目标使用青色完整轮廓，不启用面动画
- 已移除描边透明度选项；鼠标指向描边固定为完全不透明，两类宽度默认值均为 2，并使用整数步进

### v1.2.1 — 全息放置预览（2026-07-21）

- 新增全息投影实时放置预览，帮助玩家更准确地放置方块
![Facet 全息投影实时放置预览](https://github.com/user-attachments/assets/6ea970c8-b516-489e-8054-4757b45e99bf)

### v1.1.2 — 客户端涂鸦（2026-07-19）

- 新增四种客户端涂鸦标记，可用于施工和探险路标
- 新增按 `G` 打开的动画轮盘，支持数字键和鼠标选择
- 新增按世界和维度区分的本地存储；其他玩家无法看到涂鸦
- 新增自动清理：被标记方块移动、摧毁或替换后移除对应涂鸦
- 修复在 Minecraft 26.2 中打开或关闭涂鸦轮盘时短暂闪屏的问题

### v1.0.0 — 方块计数基础（2026-07-13）

- 新增可切换的方块材质描边，同时保留资源包纹理
- 新增目标方块霓虹描边，交互范围外的目标也能继续标识
- 新增距离 HUD，显示 X、Z、Y 和总曼哈顿距离
- 新增按颜色区分路径段的曼哈顿路径指示器
- 新增描边透明度、宽度和目标方块描边样式等客户端设置

## 按键绑定

- 全体方块描边开关
- 目标方块描边开关
- 方块距离计数 HUD 与曼哈顿路径显示开关
- 打开涂鸦轮盘（默认为 `G`；使用 `1`–`4` 或鼠标选择标记）
